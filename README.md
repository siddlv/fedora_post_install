# fedora_post_install

A shell script for enabling RPMFusion repositories, installing multimedia codecs and enabling flatpak on Fedora.

Usage:    
1. Open Terminal in the folder where the script is downloaded.    

2. Run the following command:    
      chmod 755 fedorapostinstall.sh

3. Run the script:    
      ./fedorapostinstall.sh

References:
https://rpmfusion.org
https://flatpak.org/setup/Fedora/
