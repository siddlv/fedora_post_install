# fedora_post_install

A shell script for enabling RPMFusion repositories, installing multimedia codecs and enabling flatpak on Fedora.

Usage:    
1. Download the script:       
            curl https://raw.githubusercontent.com/siddlv/fedora_post_install/main/fedorapostinstall.sh

2. Open Terminal in the folder where the script is downloaded.    

3. Run the following command:    
            chmod +x fedorapostinstall.sh

4. Run the script:    
            ./fedorapostinstall.sh

References:  
https://rpmfusion.org  
https://flatpak.org/setup/Fedora
